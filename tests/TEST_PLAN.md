# 测试方案（Test Plan）

依据 `Title.txt` 题目要求逐条设计。测试按**步骤**执行，每个步骤包含若干**监测节点（Node）**，每个节点输出 `PASS/FAIL` 并写入日志 `tests/logs/`。

执行入口：`bash tests/run_tests.sh`
日志输出：`tests/logs/*.log` + 汇总 `tests/logs/summary.log`
界面检测：`bash tests/run_tests.sh` 完成后，截图见 `screenshots/`，人工对照本方案"界面检测"小节。
